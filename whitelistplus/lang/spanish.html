Whitelist+

Comando Principal

El comando principal /whitelist que muestra los siguientes subcomandos:

  • /whitelist add [playerName] › Añade un jugador a la whitelist
  • /whitelist remove [playerName] › Elimina un jugador de la whitelist
  • /whitelist on › Activa la whitelist
  • /whitelist off › Desactiva la whitelist
  • /whitelist list › Muestra los jugadores que estan en la whitelist y sus nombres
  • /whitelist reload › Recarga el archivo de configuracion

Permisos

Command Permission: whitelistplus.commands.use

Bypass Permission: whitelistplus.bypass

API

The API esta divida en 8 metodos

  • WhitelistAPI.addPlayer() › Permite añadir a un jugador a la whitelist [String]
  • WhitelistAPI.removePlayer() › Permite remover a un jugador a la whitelist [String]
  • WhitelistAPI.isWhitelist() › Permite saber si un jugador esta en la whitelist [String]
  • WhitelistAPI.setWhitelist() › Permite cambiar el estado de la whitelist [Boolean]
  • WhitelistAPI.getPlayers() › Permite ver los jugadores que estan en la whitelist [String]
  • WhitelistAPI.isEnabled() › Permite obtener el estado actual de la whitelist [Boolean]
  • WhitelistAPI.setWhitelistMessage() › Permite cambiar el mensaje de expulsion cuando no estas en la whitelist [String]
  • WhitelistAPI.getAPIVersion() › Permite comprobar la version de la API [Double]
  • WhitelistAPI.getVersion() › Permite comprobar la version de la plugin [Double]
